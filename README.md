# English Training (speaking2)

AIを活用した英語スピーキング・トレーニング・アプリケーションです。

## 主な機能
- **AI会話生成**: シチュエーションに応じた英会話の練習。
- **リアルタイム添削**: 入力した英文に対する詳細な添削と解説。
- **表現の提案**: より自然な表現や、異なるトーン（フォーマル、カジュアル、シンプル）での言い換えを提案。
- **英訳練習モード**: 表示された日本語を英語に訳す練習機能。
- **Q&A機能**: システムの英文や添削内容についてAIに直接質問が可能。
- **音声読み上げ**: 高品質なニューラル音声による読み上げ（速度調整可能）。

### 復習ブックマーク時のQ&A履歴保存機能の追加
- 会話アイテムを復習に保存（ブックマーク）する際、そのアイテムに対して行った**Q&A（AIへの質問）の履歴も一緒に保存**されるようになりました。
- 復習セクションでアイテムを開いた際、過去にどのような質問をしてどのような回答を得たかをいつでも確認でき、より深い学習が可能になります。
- 復習モード中に新しく行った質問も、自動的に保存済みのデータに追記されます。

### 復習機能（Review Mode）の実装
- **練習/復習モードの切り替え**: ページ上部に「練習」と「復習」を切り替えるタブを設置しました。
- **お気に入り保存機能**: 各会話アイテムに保存ボタンを追加しました。気になったフレーズや練習した内容を「復習」セクションに保存できます。
- **復習セクション**: 保存したアイテムが一覧表示され、いつでも練習をやり直すことができます。
- **過去添削履歴の閲覧**: 復習セクションでは、練習セクションで生成した過去の回答や添削結果を「過去履歴ボタン」からいつでも確認できます。
- **継続的な学習**: 復習セクションで新たに行った練習内容も、自動的に履歴に追加保存されます。
- **復習特化UI**: 復習セクションでは、デフォルトで答え（英文）を隠した状態からスタートし、自力で思い出すトレーニングに集中できるデザインを採用しました。
- **デザインの統一とニューモーフィズムの徹底**: 復習セクション、履歴、バリエーション生成結果のすべてのUIパーツにおいて、アプリ全体のニューモーフィズム・デザインを完全に統一しました。ボタンサイズや影の質感を揃え、どのセクションでもプレミアムな操作感を提供します。

## 最近の更新
### 添削内容の要点まとめ機能の追加
- 添削コメント（Correction）の最後に、その回の添削で重要だったポイント（自然な言い回しのコツや注意点など）を、**箇条書きで簡潔にまとめる**機能を追加しました。
- 長い解説文を読まなくても、パッと見て「何を直せばよかったのか」が理解できるようになり、復習の効率が向上します。
- 会話モード・翻訳モードの両方に適用されます。

### 「伝わるか」の観点を添削に追加
- 文法的な正確さだけでなく、**実際の会話でネイティブに意図が伝わるかどうか**の観点でも添削するよう改善しました。
- 文法が不完全でも意図が読み取れる場合は「文法的には誤りがあるが意味は通じる」と評価します。
- 逆に、単語の羅列やキーワード不足で何を言いたいのか読み取れないレベルの場合は、文法の指摘より優先してその旨を指摘します。
- 会話モード・翻訳モードの両方に適用されます。

### 日本語の意図（翻訳元の意味）入力機能の追加
- **意図の伝達**: 英文を入力する際、あわせて「どのような意味のつもりで入力したか」の日本語を記入できるようになりました。
- **正確な添削**: AIがユーザーの意図を把握した上で添削を行うため、単なる文法修正にとどまらず、「言いたいことに最適な表現」をより正確に提案できるようになりました。
- **ヒントからの自動入力**: 提示された「例えば…（ヒント）」をクリックすると、その日本語訳が自動的に日本語入力欄にコピーされるため、手軽に意図を指定して練習できます。
- **学習効率の向上**: 日本語と英語をセットで意識することで、特定の状況で使うフレーズの定着を助けます。

### 設定パネルの操作性向上
- 設定項目が増えた場合でも、パネル内をスクロールしてすべての項目にアクセスできるよう改善しました。
- モバイル端末などの小さな画面でも快適に操作できるよう、スクロール領域の追加と下部セーフエリアの余白調整を行いました。
- アプリ全体のスクロールバーを、ニューモーフィズムのデザインに合わせた細身でスタイリッシュなデザインに刷新しました。

### シチュエーション選択機能の追加
- 生成される会話のシチュエーションをカテゴリー別に選択できるようになりました。
- 「日常生活」「旅行・観光」「ビジネス」「感情・表現」「社交・交流」「アカデミック」といったカテゴリーから、練習したい内容に合わせて自由に切り替えが可能です。
- デフォルトは「全選択（すべてON）」となっており、設定パネルから一括選択・一括解除も可能です。
- UIにはプレミアムなニューモーフィズム・トグルボタンを採用し、直感的な操作と視覚的な楽しさを両立しました。

### 提示済みの英文を考慮した添削ロジックの改善
- 添削時に、それまでに提示された「回答例」や「バリエーション」と同じ英文をユーザーが使用した場合、それを**間違いとして指摘しない**ように改善しました。
- 提示済みの完璧な英文を使った場合は、基本的に「正解」として扱い、必要に応じて「さらなるブラッシュアップ案」や「別の表現」として提案するようにAIへの指示を強化しました。
- これにより、お手本の英文を練習として使っているのに間違いのように指摘されるストレスが解消され、より建設的なフィードバックが得られるようになりました。

### 提案レイアウトの改善
- 提案内容（Suggestions）の「バリエーション」セクションを、元の案の右側ではなく**案の下側**に配置するように変更しました。
- これにより、スマートフォンのような幅の狭いデバイスでもバリエーションが自然に表示され、読みやすさが向上しました。

### 再挑戦（Retry）機能の実装・改善
- 添削結果の最後に「再挑戦」ボタンを追加しました。
- 添削内容を確認した後、ボタンを押すと入力がクリア（または以前の回答を再セット）され、解説を隠して再度回答に挑戦できます。
- **メイン会話での改善**: メインの入力欄（user-input）でも再挑戦が可能になりました。再挑戦ボタンを押すと、それ以降に進んだ会話（相手の返答など）が自動的に削除され、同じ状況から回答をやり直すことができます。
- 再挑戦時の添削は、**「最初の回答」や「前回の添削内容」を踏まえた内容**になります。同じ間違いを繰り返していないか、指摘されたポイントを改善できているかなど、より学習効果の高いフィードバックが得られます。
- **レイアウトの改善**: 質問セクション（item-qa-section）をフィードバックエリアの添削結果のすぐ下に、再挑戦ボタンをそのさらに下（最下部）に配置するように調整しました。これにより、一通りの添削と質問を終えた後に再挑戦を促す、より自然な学習フローになりました。

### 提案例文・回答例の動的な生成改善
- 添削後の提案例文および初期の回答例について、固定数（3個）ではなく1〜5個の間で文脈に応じて最適な数を作成するように改善しました。
- 提案内容が少ない場合に重複した内容が並ばないよう、またバリエーションが豊富な場合にはより多くの選択肢を提示できるよう、AIへの指示を最適化しました。

### デフォルト設定の変更
- アプリケーション起動時の初期設定値を最適化しました（再生速度: 0.8x、英文レベル: 簡単、生成文字数: 20文字）。

### バリエーション生成機能のUI刷新
- 「バリエーション」セクションにニューモーフィズムデザインを適用し、視覚的な一貫性を向上させました。
- 各バリエーションタイプ（フォーマル、カジュアル、簡単）にアイコンを追加し、直感的に選択できるよう改善しました。
- 生成結果の表示スタイルをプレミアムなカード形式にアップデートし、個別の再生ボタン（ micro-animation付き）を実装しました。

### 英訳練習モードのUI改善
- 英訳練習モード（btn-practice）が有効な間、元の英文（答え）を自動的に非表示にするように改善しました。これにより、答えを見ずに自力で考えるトレーニングが可能になりました。
- **英訳ボタン（btn-translate）をクリックした際、練習モードが有効であれば自動的に解除し、英訳を表示するように改善しました。これにより、練習モードを手動で閉じなくても英訳を確認できるようになりました。**

### システムの英文に対するQ&A機能の追加
- 相手（AI）が提示した英文に対しても、直接質問ができるようになりました。文法や使い方の疑問をその場で解消できます。

### 添削時のローディング表示の追加
- 英文記入後、AIが添削を生成している間に「添削中...」というメッセージとローディングアニメーションを表示するように改善しました。これにより、処理が進行中であることが明確になりました。

### AIの話し方・生成英文レベル設定の追加
- 設定パネルから、AIの回答や解説の文体（ですます調、フレンドリー、女子高生風）を選択できるようになりました。
- **女子高生スタイル（JKスタイル）について、過度な褒め言葉や持ち上げすぎを抑え、より建設的なアドバイスを重視するようにプロンプトを調整しました。**
- デフォルトで生成される英文の難易度やトーン（ネイティブ、フォーマル、カジュアル、簡単）を事前に設定できるようになりました。これらの設定はブラウザに記憶されます。

### 英文レベル「ネイティブ」の刷新とバリエーション追加
- 英文レベルの選択肢およびバリエーション生成機能に、より「生きた英語」を追求した**ネイティブ表現**を強化・追加しました。
- 単に文法的に正しいだけでなく、実際の現地の日常会話で使われるような、自然で人間味のある（非ロボット的な）表現を優先的に生成・提案するようにプロンプトを最適化しました。
- 添削時の提案（Suggestions）においても、選択したレベルに応じた自然な言い回しが提示されるようにロジックを統合しました。
- バリエーション生成メニューにも「ネイティブ」ボタンを追加し、元の文章をよりこなれた現地風の表現に一クリックで変換できるようになりました。

### UI・デザインの改善
- **保存ボタンの色修正**: 保存ボタン（btn-save）がアクティブな状態（保存済み）のときに緑色になっていた仕様を、他のボタンと同じ青色（$primary-color）に統一しました。これにより、UI全体の一貫性と美しさが向上しました。
